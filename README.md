## Проект: Место

Сайт с возможностью выкладывать фотографии мест, которые вы посетили или хотели бы посетить, добавлять им подписи и ставить лайки.

**Стек технологий:**
1. HTML
2. CSS
3. JavaScript
4. React.js
5. Figma
